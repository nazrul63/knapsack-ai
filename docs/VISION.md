# KNApSAcK-AI — Research Vision

## The Core Opportunity

The KNApSAcK Family Database contains 20+ years of curated plant metabolite
data: compound structures, biological activities, species linkages, geographic
origins, and traditional medicine formulas. This integrated multi-layer linkage
is unique — no other open database connects all these dimensions simultaneously.

The core differentiator is not raw compound count, but the integrated
**species ↔ metabolite ↔ activity ↔ formula ↔ geography** graph in one place.

## The Problem

Despite this richness:
- ~95% of metabolites have no bioactivity label (the "bioactivity gap")
- No open ML tools exist trained specifically on this dataset
- Industry cannot easily query or exploit the database without domain expertise
- Traditional medicine knowledge (KAMPO/JAMU) is largely inaccessible to AI systems

## The Vision

Build a suite of AI tools that transforms KNApSAcK from a reference database
into an active discovery engine — predicting bioactivity, validating claims,
detecting adulteration, mapping IP opportunities, and ultimately answering
natural language queries across all domains.

## Four Target Sectors

1. **Pharmaceutical / Drug Discovery** — bioactivity prediction, docking
2. **Nutraceutical / Functional Food** — regulatory claim validation, ingredient profiling
3. **Agriculture / Biopesticides** — eco-friendly pesticide candidate identification
4. **KAMPO / JAMU Traditional Medicine** — formula authentication, synergy modeling

## Academic Contribution

Each project produces a peer-reviewed publication. The capstone (KNApSAcK-GPT)
integrates validated outputs from all earlier projects into a knowledge graph
+ RAG system — publishable as a systems paper.
